pypop |Build Status|
====================

A python library to compose pop songs automatically.

Installation
------------

.. code:: bash

    # Use python3.6 or above
    $ python3 setup.py install

Example
-------

.. code:: bash

    $ python3 example.py

Supported Python Versions
-------------------------

-  3.6 and above
